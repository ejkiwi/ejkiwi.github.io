- **세부 목표**:
    - 음성 데이터 전처리 방법 익히기

    - 모델 성능 최적화 기법 학습 및 적용

### 1. 음성 데이터 처리의 전체 파이프라인

```
원본 음성 → 전처리 → MFCC 변환 → Zero Padding → 모델 입력
```

### 2. CNN과 ResNet의 차이
- **CNN**: 기본적인 합성곱 신경망
- **ResNet**: Skip Connection으로 깊은 네트워크 학습 가능

### 3. 모델 성능 최적화 기법
- 데이터 증강
- 적절한 손실 함수와 옵티마이저 선택
- 정규화 기법 (Dropout, Batch Normalization)
- 하이퍼파라미터 튜닝

### 4. 디버깅
- 레이블 인코딩 문제 해결
- 과적합 문제 인식 및 대응
	- **문제**: 훈련 데이터에서는 잘 작동하지만 실제 정확도 3%  
	- **해결**: 모델 구조 변경, 정규화 기법 적용, 하이퍼파라미터 조정
- PyTorch 버전 호환성 문제 해결
- 레이블 인코딩 문제
	- **문제**: 문자열 레이블로 인한 훈련 오류
	- **해결**: 레이블 매핑 딕셔너리 생성하여 숫자로 변환
