2차 학습동아리에 참여한 이유

24년 여름방학 모각코(모여서 각자 코딩) 활동에서 처음 만나 각자 인공지능 기초 공부를 진행했었는데, 이번에는 한 단계 더 나아가 실제로 음성 데이터를 다루는 인공지능 모델을 직접 구현해보자는 목표를 가지고 또 참여하게 되었다.

목표

  • 최종 목표: 여러 최적화 방법을 도입하여 성능을 높인 음성 분류 모델 구현
  • 세부 목표:
    • 음성 데이터 전처리 방법 익히기
    • CNN, ResNet 등 다양한 딥러닝 모델 실습
    • 모델 성능 최적화 기법 학습 및 적용
    • 최대한 높은 정확도 달성

배운 것들, 해본 것들, 느낀 점

1. 음성 데이터 처리의 전체 파이프라인

원본 음성 → 전처리 → MFCC 변환 → Zero Padding → 모델 입력

2. CNN과 ResNet의 차이

  • CNN: 기본적인 합성곱 신경망
  • ResNet: Skip Connection으로 깊은 네트워크 학습 가능

3. 모델 성능 최적화 기법

  • 데이터 증강
  • 적절한 손실 함수와 옵티마이저 선택
  • 정규화 기법 (Dropout, Batch Normalization)
  • 하이퍼파라미터 튜닝

4. 디버깅

  • 레이블 인코딩 문제 해결
  • 과적합 문제 인식 및 대응
    • 문제: 훈련 데이터에서는 잘 작동하지만 실제 정확도 3%
    • 해결: 모델 구조 변경, 정규화 기법 적용, 하이퍼파라미터 조정
  • PyTorch 버전 호환성 문제 해결
  • 레이블 인코딩 문제
    • 문제: 문자열 레이블로 인한 훈련 오류
    • 해결: 레이블 매핑 딕셔너리 생성하여 숫자로 변환

느낀 점

지금 생각해보면 예제 코드에 많이 의존했던 점이 아쉽다. 좀 더 스스로 구현해보고 싶었지만 역량이 많이 부족했던 것 같다.
실습 활동을 하면서 생각보다 오류가 많이 났고, 생각보다 스스로 그 오류를 해결할 수 있는 힘이 있음을 느낄 수 있었다. 그동안 인공지능 공부가 내가 하기에 엄청 어렵다는 생각에 주저했으나 그동안 내 의지의 문제였던 것 같다.
